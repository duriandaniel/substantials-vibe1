─────────────────────────────────────────────
STEP 1 — scraper.py
─────────────────────────────────────────────
Fetch today's ASX announcements and download substantial holder PDFs.

1. GET https://www.asx.com.au/asx/v2/statistics/todayAnns.do
2. Parse the HTML table with BeautifulSoup
3. Filter rows where headline contains "substantial" (case-insensitive)
4. For each match, extract:
   - asx_code (ticker column)
   - headline
   - announcement_id (idsId parameter in the PDF link)
   - lodgement_date (today's date)
   - pdf_url: construct as https://www.asx.com.au/asxpdf/{YYYYMMDD}/pdf/{idsId}.pdf
5. Skip any announcement_id already in processed_ids.txt
6. Download new PDFs to pdfs/{announcement_id}.pdf

Test: run scraper.py standalone and confirm PDFs appear in pdfs/

─────────────────────────────────────────────
STEP 2 — parser.py
─────────────────────────────────────────────
Two-tier parser as described in CLAUDE.md.

Tier 1 (pdfplumber):
- Extract full text from PDF
- Regex search for: investment_manager, previous_percent, new_percent, date_of_change
- All 4 found → confidence="high", parse_method="rule-based", return result
- Any missing → go to Tier 2

Tier 2 (Claude API):
- Model: claude-sonnet-4-20250514
- Send extracted text with the system prompt from CLAUDE.md
- Parse JSON response strictly
- Fields present → confidence="low", parse_method="ai"
- Still missing required fields → confidence="needs_review"

Test: run parser.py against every PDF in test_pdfs/ and print results table.

STEP 4 — notifier.py
─────────────────────────────────────────────

- Use env vars: ALERT_EMAIL_FROM, ALERT_EMAIL_TO, SMTP_PASSWORD
- Port 587, STARTTLS
- Include: announcement_id, asx_code, pdf_url, confidence, missing fields

─────────────────────────────────────────────
STEP 5 — main.py
─────────────────────────────────────────────
Orchestrate all modules. Core loop:

  check trading day → exit cleanly if not
  announcements = scraper.get_announcements()
  for each announcement:
    try:
      download pdf
      parse pdf
      write to csv
      send alert if confidence != "high"
      mark as processed
    except → log and continue (NEVER crash full pipeline)

Test: python main.py end to end → output.csv populated.
